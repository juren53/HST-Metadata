### How To Download a File from GitHub

The following describes the steps to dowload individual files located in the HST-Medtadata GitHub Repository.

1. Navigate to the GitHub repository page that contains the file. [ Python files for HST Metadata Processing are located at https://github.com/juren53/HST-Metadata/tree/master/Code ]
2. Click the file you want to download. This will open the file in GitHub.
3. On the file page, click the Raw button. This opens the raw code in your browser.
4. Right-click the page and select Save as…, then name the file and choose where to save it on your device.
