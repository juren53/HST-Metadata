# HST-Metadata


The purpose of this project is to adapt and build tools that can embed metadata in media files at the Harry S. Truman Presidental Library i.e. photos, audio and video files. 

The project consists of three phases. The initial focus is the photo collection to be followed by audio files and finally video files at te Library.

This work is under the direction of Laurie Austin, Audiovisual Archives, Harry S. Truman Presidential Library.
