Configuring nomacs image viewer

for

HSTL Metadata Post-processing Review




The document will describe how to configure the nomacs image viewer to enable the rapid review of HSTL images with metadata tags for post-processing review.


Below is the nomacs user interface “out-of-the-box” with just the image displayed in the middle of the screen.










To begin configuring the nomacs image viewer, select Panel from the top menu and click on the File Explorer and Metadata Info options in the pull down menu. This will create a three panel interface with the File Explorer on the left, the Image in the middle and the Metadata Info on the right as shown below.

















Next, the Metadata Info panel on the left can be “cleaned up” to display just the IPTC metadata that was recently added to the image.








Note: the focus needs be established on the image panel to enable the paging through the collection of images using the left and right arrow keys. The focus is established by clicking on the image panel.
